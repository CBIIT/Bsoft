685  Measures the astigmatism between the resolution limits
686  The variable real_size is the box x-size in angstrom
687  Zeroes are calculated, given in spatial frequency units
688  For every two successive zeroes between the resolution limits:
689  the minimum within a kernel is found at each zero: [ib1]:b1 and [ib2]:b2
690  the maximum halfway between the zeroes within a kernel is found: [im]:m

692  the intent is that this measure should be maximum for the correct astigmatism parameters
