57 /*
58 @brief Functions for particle picking.
59  Box interface syntax:
60  Bmg box [item] [action] [params]
61  Item: Must contain "Micrograph" or "Reconstruction"
62  Actions:
63  count [n] [|"bad"] counts the number of particles or bad areas
64  ids [fc|"bad"] [fi] returns the identifiers of particles or bad areas
65  location [id] returns the location of a particle or bad area
66  fom [id] [fi] returns the fom of a particle with the indicated index
67  fom_min returns minimum FOM value
68  fom_max returns maximum FOM value
69  select [id] [s] sets the selection number of a particle
70  select [x] [y] [z] returns the identifier at the given location
71  select_min returns minimum selection value
72  select_max returns maximum selection value
73  move [id] [dx] [dy] [dz] updates a particle location by the given vector
74  create [x] [y] [z] [n] [t] [s] creates a new particle at the given location
75  renumber [n] renumbers particles
76  delete [all|mg|rec|id] [n] deletes particles or bad areas
77  template updates picked template
78  pickcc [hr] [lr] [fmn] [fmx] [ed] [b] picks particles by cross-correlation with a template
79  pickvar [ak] [vk] [ed] [b] [ns] picks particles from a variance map
80  extract [f] [of] [bf] [nf] [ft] [fv] [mw] [sf] [p] extracts particles from a micrograph
81  Parameters:
82  id identifier
83  t box type: 1=particle, -1=bad area
84  fc FOM cutoff
85  fmn FOM minimum
86  fmx FOM maximum
87  fi FOM index (0-4)
88  s selection number
89  x x location
90  y y location
91  z z location
92  n image number
93  dx change in x
94  dy change in y
95  dz change in z
96  r particle radius
97  e gaussian edge for picking particles
98  hr high resolution limit
99  lr low resolution limit
100  ak averaging kernel edge size
101  av variance kernel edge size
102  ed exclusion distance for picking particles
103  b binning for picking particles
104  ns number of standard deviations for picking particles
105  f file name
106  of flag indicating odd box size
107  bf background correction flag
108  nf normalization flag
109  ft fill type
110  fv fill value
111  mw filament mask width
112  sf split flag
113  p particle path
114 */
