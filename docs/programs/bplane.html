Bsoft program

bplane

Usage: bplane [options] in.star

Generates and manipulates plane models.

Actions for creation:

-plane 386,125Generate a plane with the given length and width.
-setviewsFit a plane to the model and set the component views.
-fitUse the input model as guide to generate a plane model.

Actions:

-view 0.5,-0.2,0.3,35View to orient plane: vector=normal, angle=in-plane rotation.
-guide poly.starInput guide model to adjust the plane to.

Parameters:

-verbose 7Verbosity of output.
-separation 14.5New component separation (default 10).
-sigma 150Smoothness of the fit (default 100).
-componentradius 0.5Component display radius.
-linkradius 0.5Link display radius.

Input:

-map density.mapInput map file.

Output:

-output newmod.starOutput model file.
