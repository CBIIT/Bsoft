Bsoft program

bdelta

Usage: bdelta [options] in.star [in2.star ...]

Generates and analyzes deltagraphs.
The creation options are mutually exclusive.

Actions:

-pentagonalCreate a pentagonal deltagraph.
-hexagonalCreate a hexagonal deltagraph.
-tube 8,3Create a tubular deltagraph with the given lattice parameters.
-caps 2Add 1 or 2 caps.
-dualCalculate the dual of a polyhedron.

Parameters:

-verbose 7Verbosity of output.
-componentradius 0.5Component display radius.
-linkradius 0.5Link display radius.
-linklength 150Link length (default 1).
-radius 3Radius in integer units (default 1).
-height 5Height in integer units (default 1).

Input:

-parameters param.starInput parameter file.

Output:

-output file.starOutput model parameter file.
