Bsoft program

bsplitmol

Usage: bsplitmol [options] input.pdb output.pdb

Splits sets of coordinates to different numbered files.

Parameters:

-verbose 7Verbosity of output.
-first 5Number given to the first file (default 0).
-digits 3Number of digits inserted before the last period in the output file name.

Input:

-parameters parm.starAtomic properties parameter file (default atom_prop.star).
