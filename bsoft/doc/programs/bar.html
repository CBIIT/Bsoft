Bsoft program

bar

Usage: bar [options] input.img output.img

Does simple arithmetic on images.
Operations are done in the order given on the command line.

Actions:

-add -5.78Add a constant to each voxel.
-multiply 6.123Multiply each voxel with a constant.
-power 0.75Raise to the power (must be positive).

Parameters:

-verbose 7Verbosity of output.
-datatype uForce writing of a new data type.
