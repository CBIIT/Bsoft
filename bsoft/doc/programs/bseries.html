Bsoft program

bseries

Usage: bseries [options] input.star [input.star]

Aligns series of micrographs using consecutive pairwise cross-correlation.

Actions:

-bin 3,2,1Binning by the given kernel size - one value = isotropic binning.
(i.e., 2 = binning with a 2x2 kernel, changes the units).
-align 56Align to this reference image (image numbers start at 0).
-filament 5.3,50Box particles along filaments with this interval and mask width (voxels).
-extract 150,111,120Extract particles with this size (voxels, one sets all).

Parameters:

-verbose 7Verbosity of output.
-datatype uForce writing of a new data type.
-sampling 1.5,1.5,1.5Sampling (A/pixel; a single value can be given).
-origin 110,50,44Origin for tilt axis (default 0,0,0).

Parameters for alignment:

-framesFlag to align micrograph frames.
-dose 2.5Weigh the amplitudes by accumulated dose using the given dose/frame (e/Ã…2).
-exposure 8.5Exposure time to calculate drift rate from alignment (seconds)
-rate 4.5Frame rate (frames/second)
-countsFlag to rescale images based on their counts histogram.
-resolution 900,300High and low resolution limits for cross-correlation (default 0.1,1000 angstrom).
-shiftlimit 3.5Limit on origin shift relative to nominal center (default 10% of box edge size).
-edge 23,12Smooth the edge to a given width, with gaussian decay of a given width.
-subset 2-8,12Subset of micrographs to align and average.

Parameters for particle extraction:

-fill 127Fill value for particles extending beyond image (default background).
-partbase subvolSet the particle base file name (no extension).
-partpath dir/subdirSet the particle file paths.
-partextension pifSet the particle image file format extension.
-helix 16.2,39.5Helical parameters for filament boxing: helical rise
and angle per asymmetric unit (angstroms and degrees).
-backgroundCorrect background of extracted particles.
-normalizeNormalize extracted particles.

Input:

-Gainreference gr.tifGain reference to correct the input micrographs.
-Mask mask.tifMask file to use for cross-correlation.

Output:

-output file.starOutput parameter file, if -average, also output average parameter file.
-write sumOutput either aligned frames, sum or avg.
-image file.pifOutput multi-image file with aligned images.
-average file.pifOutput average of aligned images.
