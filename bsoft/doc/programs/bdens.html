Bsoft program

bdens

Usage: bdens [options] input.img output.img

Calculate density in selected regions of a map.

Actions:

-box 12,34,2,45,34,25Calculate statistics within this box (start and size).
-oval 5,10,28,50,22,89Calculate statistics within this oval (start and size).

Parameters:

-verbose 7Verbose output.
-datatype uForce writing of a new data type.
-select 1Select an image (first image = 0; default: 0).

Input:

-Mask mask.mapCalculate statistics with a mask with any number of levels.
