Bsoft program

btree

Usage: btree [options] tree.ph tree.ps

Draws phylogenetic trees as postscript.
Handles phylip format trees as input.

Parameters:

-verbose 7Verbosity of output.
-scale 350Scale for postscript output (default 200).
-rotate 110Rotate for postscript output (default 0 degrees).
