Bsoft program

bfix

Usage: bfix [options] file.in file.out

Replaces single characters in a file.
If only one file name is given, the output is sent to stdout. If a "." character is used for output, the input file name is used (without the path). Note: replace and delete are mutually exclusive - the last option wins

Actions:

-nReplace with (To UNIX style)
-cReplace with (To MAC style)
-replace 9,32Replace all character values 9 with character values 32
-delete 13Delete all character values 13

Parameters:

-verbose 7Verbosity of output
