Bsoft program

bpath

Usage: bpath [options] in.star

Analyzes paths and cycles in a model.

Actions:

-allReset selection to all components before other selections.
-cutoff 23.5Assign links within this cutoff distance.

Parameters:

-verbose 7Verbosity of output.
-componentradius 0.5Component display radius.
-linkradius 0.5Link display radius.

Input:

-parameters param.starInput parameter file.

Output:

-output path.starOutput model file with components linked along paths.
