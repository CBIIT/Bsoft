Platforms

Binary distributions are available for Mac OSX and Linux. This is the preferred way of installing Bsoft

Bsoft can be compiled from the source on any Unix platform. The following has been tested in the past:

  • Linux (i386/i686, x86_64, PowerPC, Alpha, Itanium)
  • Darwin or MacOSX (PowerPC, Intel)
  • IRIX64 (SGI)
  • Solaris (Sun)
  • AIX (IBM)
  • OSF1 or Tru64 (Alpha)

Note that compiling Bsoft 2 is different from the previous versions and has not been tested on all systems. It also contains new features added to C++ that requires recent compilers.
