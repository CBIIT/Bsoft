JSON in Bsoft

JSON (Javascript object notation) is a native format on the web. It has a text-based file format that can have a conceptual internal database-like format in code. I introduced it into Bsoft to have a consistent and flexible way of dealing with image and processing parameters. It is hierarchical (like XML) and allows for sophisticated search and reorganization. These files have the ".json" extension.
