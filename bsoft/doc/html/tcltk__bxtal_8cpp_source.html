40 @brief Functions for 2D crystallographic indexing.
41  Diffraction spot interface syntax:
42  Bmg spot [item] [action] [params]
43  Actions:
44  count counts the number of spots
45  ids returns the spot indices (3 per spot)
46  reindex [h1] [k1] [l1] [h2] [k2] [l2] re-indexes a spot
47  location [h] [k] [l] returns the location of a spot
48  fom [h] [k] [l] returns the FOM of a spot
49  select [x] [y] [z] returns the spot indices at the given location
50  move [h] [k] [l] [dx] [dy] [dz] updates a spot location by the given vector
51  create [h] [k] [l] [x] [y] [z] creates a new spot at the given location
52  delete [all|mg|rec|h] [k] [l] deletes spots
53  unitcell calculates unit cell vectors
54  generate [m] generates spots
55  mask [r] masks spots
56  Parameters:
57  h k l Miller indices
58  x x location
59  y y location
60  z z location
61  dx change in x
62  dy change in y
63  dz change in z
64  m limit in reciprocal space for generating spots
65  r mask radius
66 */
