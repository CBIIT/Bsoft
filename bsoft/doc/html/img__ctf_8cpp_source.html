772 /*
773  Expects a radial power spectrum
774  Measures the astigmatism between the resolution limits
775  The variable real_size is the box x-size in angstrom
776  Zeroes are calculated, given in spatial frequency units
777  For every two successive zeroes between the resolution limits:
778  the minimum within a kernel is found at each zero: [ib1]:b1 and [ib2]:b2
779  the maximum halfway between the zeroes within a kernel is found: [im]:m
780  Measure: sum(im*(m - (b1+b2)/2))
781  the intent is that this measure should be maximum for the correct astigmatism parameters
