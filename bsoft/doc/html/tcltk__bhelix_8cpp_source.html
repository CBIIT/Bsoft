43 @brief Functions for helical indexing.
44  Layer line interface syntax:
45  Bmg layerline [item] [action] [params]
46  Actions:
47  count counts the number of layer lines
48  ids returns the identifiers of the layer lines
49  location [id] returns the location of a layer line
50  fom [id] returns the figure-of-merit of a layer line
51  select [x] [y] returns the layer line identifier at the given location
52  move [id] [dx] [dy] updates a layer line location by the given vector
53  create [x] [y] [id] creates a new layer line at the given location
54  delete [all|mg|id] deletes layer lines
55  order [id] [dx] [dy] sets or returns the order of a layer line
56  generate [m] generates all layer lines
57  mask [w] masks layer lines
58  plot [id] [l] retrieves one layer line's values as an array of length l
59  bessel [o] [r] [l] retrieves a bessel function of given order and radius as an array of length l
60  Parameters:
61  id layer line identifier
62  x x location
63  y y location
64  dx change in x
65  dy change in y
66  m index limit
67  w mask width
68  l array length
69  o Bessel order
