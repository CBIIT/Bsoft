64 /************************************************************************
65 @Object: struct Bstar
66 @Description:
67  Overall STAR structure to hold all data blocks and options for I/O..
68 @Features:
69  The split flag allows the user to output data blocks in separte files
70  in stead of one big file.
71  The line length field allows the user to output long lines without
72  wrapping it around.
73  The comments are ignored but output to the a new file - this can be used
74  to document the history of the file.
75  The STAR database is a hierarchy consisting of blocks, each with a set
76  of items.
77 *************************************************************************/
78 struct Bstar {

80  int line_length; // Length of lines in output file
