11/************************************************************************
12@Object: struct Bnode
13@Description:
14 Unrooted tree node structure.
15@Features:
16 An unrooted tree consists of a set of trifurcated internal nodes and
17 a set of end or tip nodes. The choice of the root node is arbitrary,
18 although it is always defined as the entry point into the tree.
19 The branch length is the distance from the node closer to the root
20 of the tree (thus inherent directionality which must be reset if
21 the tree is rerooted). The plot angle is just a convenient field
22 for tree display.
23*************************************************************************/
