Bmatrix read_matrix ( Bstring &  filename)

Reads a matrix file.

The matrix file contains the keyword "matrix_", following by a line with column labels and multiple lines for the matrix contents. All elements are delimited by whitespace. The matrix is packed into a data array row by row. Requirements: Only square matrices are supported, i.e., there must be as many content lines as there are column labels.

Parameters
Bstring&filename matrix file name.
Returns
Bmatrix a pointer to the matrix.
