Selects reflections for output.

Parameters
*pcomplex image.
hireshigh resolution limit.
loreslow resolution limit.
coneexclude missing cone data, set to PI/2 for none excluded.
weighflag: if 1 then multiply amplitude with FOM.
fomcutFOM cutoff value.
Returns
unsigned long number of reflections selected. 
Reflections or structure factors are selected based on:
1.  not in the missing cone.
2.  above the FOM cutoff value.
The reflections may be optionally weighed with the FOM.
